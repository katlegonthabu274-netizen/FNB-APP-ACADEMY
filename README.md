#### student_info.py
**Description:**
A Python program that collects a student's personal information and displays it as a formatted profile card. The program demonstrates:

- Arithmetic calculations
- The `round()` function
- The `type()` function

  ## Unit 2 – Username and Message Formatter

### Description
This Python program collects a user's first name, last name, and a short bio. It then formats the user's information by creating a username, cleaning the bio, modifying text, and displaying a formatted user profile.

### Learning Outcomes
- Accept user input using `input()`
- Create and store variables
- Use string indexing to access individual characters
- Convert text to lowercase using `.lower()`
- Convert names to Title Case using `.title()`
- Remove leading and trailing whitespace using `.strip()`
- Replace text using `.replace()`
- Count characters using `len()`
- Create formatted strings using f-strings
- Display a structured user profile

### Skills Demonstrated
- User Input
- String Manipulation
- String Indexing
- Variables
- Built-in Functions
- String Methods
- Formatted Output (f-strings)

### Files
- `string_formatter.py` – Formats user information into a clean and structured profile.
- Formatted output using f-strings
