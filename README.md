# FNB App Academy

This repository contains my Python exercises, practical tasks, and projects completed during the FNB App Academy.

## Unit 1 – Python Fundamentals

### Learning Outcomes
- Variables
- Data Types
- User Input
- String Methods
- Arithmetic Operations
- f-Strings

### Files

#### hello.py
**Description:**
A simple Python program that prints a welcome message to demonstrate the use of the `print()` function.

#### student_info.py
**Description:**
A Python program that collects a student's personal information and displays it as a formatted profile card. The program demonstrates:
- User input (`input()`)
- Data types (`str`, `int`, `float`)
- String methods (`.upper()`, `.title()`)
- Arithmetic calculations
- The `round()` function
- The `type()` function
- Formatted output using f-strings
